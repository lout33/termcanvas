# v0.2.0

> Released: 2026-04-08

## Highlights

- **Active canvas terminal sub bar** - The canvas topbar now gives active-terminal context directly in navigation chrome, making it easier to understand which terminal you are working in while switching canvases.
- **Fullscreen keeps workspace panels available** - Maximized terminals now behave like a classic editor workspace: the left drawer and right preview panel stay usable instead of disappearing behind fullscreen terminal mode.
- **Refined app branding and topbar polish** - The app icon, brand mark, and topbar navigation were refreshed so the desktop shell feels more intentional and easier to scan.

## Improvements

- Maximized terminals now respect the visible workspace insets from the sidebar and file inspector.
- The sidebar toggle remains available while the drawer is open and now shows a clearer open/close icon state.
- Both side panels now live below the top header instead of occupying app-level chrome.

## Fixes

- Fixed the fullscreen terminal layout so opening workspace chrome no longer forces the sidebar out of reach.
- Fixed panel structure regressions with targeted tests for sidebar and inspector placement.
